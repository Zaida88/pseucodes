La declaración if

Muy a menudo, cuando escribe código, desea realizar diferentes acciones en función de diferentes condiciones. Puede hacer esto usando declaraciones condicionales en su código.
Use if para especificar un bloque de código que se ejecutará si una condición especificada es verdadera.



Las declaraciones se ejecutarán solo si la condición especificada es verdadera.
